# 安那其主义书库

## 介绍

本仓库主要保存一些安那其主义以及与安那其主义相关的一些中文书籍，我将会按照作者将著作进行分类。安那其主义（Anarchism）即无政府主义，是一门政治理念及运动，它反对所有强制性的阶级制度。安那其主义认为国家对人民有害，本身没有存在的必要。

国内关于安那其主义的介绍，相关著述引进也不多，所以将为数不多的中文著述集中在一起也是迫切的。

## 推书

本人能力有限，不能囊括所有中文的安其那书籍，如有遗漏的欢迎补充。
